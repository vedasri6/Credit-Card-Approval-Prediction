Aegis REST API Integration

Aegis credit prediction engine is accessible via standardized JSON REST APIs. Integrate inference operations inside client systems or microservice architectures.

API References
POST /api/predict

Submits applicant profile details in standard JSON payload formatting. Returns approval classification status, confidence intervals, risk metrics, and scoring.

JSON Body Schema Details
{
  "gender": "Male",
  "age": 30,
  "income": 5000,
  "employment_type": "Salaried",
  "employment_years": 4.0,
  "education": "Graduate",
  "marital_status": "Married",
  "dependents": 1,
  "occupation": "Engineer",
  "residence_type": "Owned",
  "credit_history": "Good",
  "existing_loans": 0,
  "payment_history": "No Delays",
  "annual_income": 60000,
  "loan_amount": 15000,
  "loan_purpose": "Home"
}
cURL Request Code Snippet
curl -X POST http://localhost:5000/api/predict \
  -H "Content-Type: application/json" \
  -d '{"gender":"Male","age":30,"income":5000,"employment_type":"Salaried","employment_years":4.0,"education":"Graduate","marital_status":"Married","dependents":1,"occupation":"Engineer","residence_type":"Owned","credit_history":"Good","existing_loans":0,"payment_history":"No Delays","annual_income":60000,"loan_amount":15000,"loan_purpose":"Home"}'
